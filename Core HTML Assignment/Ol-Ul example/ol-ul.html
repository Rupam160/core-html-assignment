Common Text Formatting Tags in HTML

  1. <strong>
    • Use to emphasize text with semantic importance.
    • Best practice: Use sparingly for important text, such as warnings or keywords.
  2. <em>
    • Used to emphasize text with stress or importance.
    • Best practice: Use when the emphasis is stylistically different from normal text.
  3. <b>
    • Originally for bold text, now used for styling purposes.
    • Best practice: Prefer <strong> for semantic meaning; use <b> sparingly for presentational purposes.
  4. <i>
    • Originally for italic text, now used for styling purposes.
    • Best practice: Prefer <em> for semantic meaning; use <i> sparingly for presentational purposes.
  5. <u>
    • Used to underline text.
    • Best practice: Avoid using for general text formatting due to accessibility and usability concerns; use CSS for underlining text.
  6. <sub>
    • Defines subscripted text, typically used for chemical formulas, mathematical expressions, etc.
    • Best practice: Use for content where subscripting is necessary, such as in scientific notation.
  7. <sup>
    • Defines superscripted text, commonly used for footnotes, exponentiation, etc.
    • Best practice: Use for content where superscripting is necessary, such as in mathematical expressions.
