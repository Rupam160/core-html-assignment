1. HTML

    Primary Use Cases Key Features/Benifits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benifits
    Styling and visually designing web pages, including layout, colors, fonts, and animations
    • Enhances the appearance and user experience of websites
    • Separates content from presentation, promoting cleaner code
    • Supports responsive design for various screen sizes and devices

  3. Java Script

    Primary Use Cases Key Features/Benifits
    Adding interactivity and behavior to web pages, enabling dynamic updates and user interactions
    • Facilitates client-side scripting for real-time responses
    • Enables DOM manipulation for dynamic content changes
    • Supports asynchronous programming for efficient handling of operations like data fetching and form submission
