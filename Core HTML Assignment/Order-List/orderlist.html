1. <html> : The root element of an HTML document, containing all other elements.
  2. <head> : Contains metadata, title, links to stylesheets, and other information for the document.
  3. <title> : Sets the title of the document, displayed in the browser tab.
  4. <link> : Used to link external resources like stylesheets and icons.
  5. <style> : Embeds CSS styles within an HTML document.
  6. <nav> : Represents a section of a page that contains navigation links.
  7. <section> : Defines a section in a document, used to group related content.
  8. <article> : Represents a self-contained composition that can be independently distributed or reused.
